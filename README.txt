Это реализация простенького приложения.

На вход подается название города, его можно подавать на русском или английском.
На выходе пользователь получает краткую информационную сводку о погоде в этом городе.

Чтобы воспроизвести данный код, советую прописать в терминале pip install requirements.txt, будут поставлены библиотеки нужной версии.
Так же потребуется зарегистророваться на OpenWeatherMap (ссылка: https://openweathermap.org/) и получить API ключ, чтобы получать информацию о погоде.
